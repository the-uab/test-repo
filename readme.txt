Ayuda para los principiantes.
1. instalar el Git
2. Crear una carpeta de trabajo
3. Click derecho y seleccionar la opción Git Bach
4. Aparecera una ventana en consola de trabajo con git
5. Para copiar los archivos del Proyecto descargar los archivos del proyecto
   https://github.com/the-uab 
   para las pruebas ingresar al link "test-repo"
   en la seccion "HTTPS clone URL" copiar la dirección
   y en la ventana escribir
   git clone (pegar la dirección del proyecto de la pagina web)
   con esto estas opteniendo los archivos del proyecto.
6. escribir git status
7. en la copia local probar (modificar el contenido y guargar) el archivo readme.txt
8. escribir git status
9. subir las modificaciones con:
   git commit -am "Mensaje del trabajo"
   (ojo leer los mensajes para cada caso)
10. escribir git status
11. actualizar al servidor github con:
	git push master  (luego pedira usuario y contraseña) luego 
	git status
12. ver en la web los cambios realizados.
13. si no funciona algo anda mal.
14. si lo lograste congratulation.
